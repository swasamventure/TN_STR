ARBORVISTA RETREAT — GITHUB PAGES V3.5

This package combines the planned Version 3 and Version 4 enhancements:
- Full-image-fit cinematic hero slideshow
- Filterable masonry gallery
- Fullscreen lightbox with previous/next controls
- Interactive trip planner
- Live Sevierville weather and local time
- Nearby destination cards
- Airbnb embed and booking CTAs
- Responsive mobile navigation and fixed booking bar
- SEO metadata and structured data

UPLOAD INSTRUCTIONS
1. Extract this ZIP.
2. Upload the CONTENTS of the folder to the ROOT of your TN_STR repository.
3. Replace assets/, css/, js/, index.html, 404.html, robots.txt, sitemap.xml and README-UPLOAD.txt.
4. KEEP your existing images/ folder unchanged.
5. Commit the files and wait 2–5 minutes for GitHub Pages.
6. Hard refresh with Ctrl+Shift+R or open the site with ?v=35.
