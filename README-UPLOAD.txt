ARBORVISTA RETREAT — PRODUCTION RELEASE

1. Keep the existing images/ folder in the GitHub repository.
2. Upload everything in this package to the repository root.
3. Replace existing files when GitHub asks.
4. Commit the changes.
5. Wait 2–5 minutes, then hard-refresh the live site.

This release uses every currently catalogued property image in these folders:
- images/hero
- images/living
- images/kitchen
- images/dinning
- images/bedrooms
- images/bathrooms
- images/hot-tub
- images/Area Map.png

The package intentionally does not duplicate the large images folder because it already exists in GitHub.
